Strip de espacios en blanco

Es bueno conocer estos métodos ya que al momento de comparar strings un espacio en blanco influye.

'Python' != 'Python '

para eliminar lo espacios en blanco a la derecha del string usamos el método "rstrip"


favorite_language = 'python '
print(favorite_language)
# 'python '
print(favorite_language.rstrip())
# 'python'

Para eliminar los espacios en blanco del lado izquierdo usamos "lstrip"

Para eliminar los espacios de ambos lados usamos "strip"


favorite_language = ' python '
print(favorite_language.rstrip())
# ' python'
print(favorite_language.lstrip())
# 'python '
print(favorite_language.strip())
#'python'

Sobre la inmutabilidad de los strings . . .

Obviamente los métodos están siendo aplicados al momento de imprimir el string, no lo modifican ya que los strings en Python son inmutables

Si quisieramos de alguna forma "cambiar" el string debemos redefinirlo

Lo que verdaderamente ocurre es que la variable inicial esta apuntando al espacio de memoria de el string generado por los métodos de este. Estamos reasignandolo. 


favorite_language = ' python '

favorite_language = favorite_language.strip()

Listas

Básicamente las listas son un equivalente a los arreglos en otros lenguajes

Pero no son arreglos (los elementos no están necesariamente en espacios contiguos de memoria)

Son una colección de cosas en algún orden en particular


marcas_autos = ['toyota','ford','chevrolet','mercedes']
print(marcas_autos)
Al imprimir directamente una lsita podemos ver la representación en string de esta 

['toyota','ford','chevrolet','mercedes']

Accesando elementos de la lista


print(marcas_autos[0])
# toyota
print(marcas_autos[0].title())
# Toyota

Python también permite accesar los elementos desde el ultimo de esta forma:


print(marcas_autos[-1])
# mercedes
print(marcas_autos[-1].title())
# Mercedes
print(marcas_autos[-2].title())
# Chevrolet

Entonces con esto ya sabemos como usar un elemento de una lista para otras cosas . . .


print("En Venezuela los " + marcas_autos[0].title() + " son vendidos con sobreprecio")
# En Venezuela los Toyota son vendidos con sobreprecio

# tambien es posible pasar varios argumentos a print, en este caso 3 strings y el pondrá la separación
print("En Venezuela los",marcas_autos[0].title(),"son vendidos con sobreprecio")
# En Venezuela los Toyota son vendidos con sobreprecio

Usando el método pop

Este método hace referencia al "pop" que usualmente es asociado con las pilas por lo que sabemos que este nos dejará sacar de la lista el último elemento Y almacenarlo en una variable por si queremos hacer algo con el. 


marcas_motos = ['ducati','honda','yamaha','suzuki']
marca_eliminada = marcas_motos.pop()
print(marca_eliminada)
# suzuki
print(marcas_motos)
# ['ducati','honda','yamaha']

Aunque es posible que necesitemos hacer algo con otro elemento que quisieramos fuera de la lista que no necesariamente es el último, pop puede ser invocado con un atributo indicando cual es la posición que quisieramos tomar. 


marcas_motos = ['ducati','honda','yamaha','suzuki']
marca_no_japonesa = marcas_motos.pop(0)
print("La marca no japonesa es",marca_no_japonesa)
# La marca no japonesa es ducati

Ordenando una lista

Ordenando una lista permanentemente con el método sort()

Sort ordena una lista permanentemente, esto quiere decir que no tendremos la lista "inicial", pues esta "muta" para convertirse a su versión ordenada.

Técnicamente la lista no mutó sino que la variable que apunta a esta ahora apunta a una nueva versión de esta que está ordenada. 


autos = ['bmw', 'audi', 'toyota', 'subaru']
autos.sort()
print(autos)
# ['audi', 'bmw', 'subaru', 'toyota']

Podemos ordenar esta lista a la inversa pasando el parámetro opcional "reverse" como "True"


autos.sort(reverse=True)
print(autos)
# ['toyota', 'subaru', 'bmw', 'audi']

Ordenando una lista temporalmente con el método sorted()

Sorted retorna una copia de nuestra lista pero ordenada, es buena idea usarla sino queremos cambiar permanentemente el orden de nuestra lista inicial. 


autos = ['bmw', 'audi', 'toyota', 'subaru']
print(autos)
# ['bmw', 'audi', 'toyota', 'subaru']

print(autos.sorted())
# ['audi', 'bmw', 'subaru', 'toyota']

print(autos) # observemos que la lista original se mantiene intacta
# ['bmw', 'audi', 'toyota', 'subaru']

Obteniendo el reverso de una lista

Nótese que no ordena alfabéticamente en reversa los elementos sino que devuelve la lista pasada en reversa.

Reverse cambia el orden de la lista permanentemente


autos = ['bmw', 'audi', 'toyota', 'subaru']
autos.reverse()
print(autos)
# ['subaru', 'toyota', 'audi', 'bmw']

Trabajando con parte de una lista (slicing)

Para hacer un "slice" o porción de una lista debemos especificar los indices del primer y ultimo elemento con el que deseamos trabajar. Python se detiene un índice antes del último número que especificas. 


players = ['charles', 'martina', 'michael', 'florence', 'eli']
print(players[0:3])
# ['charles', 'martina', 'michael']
print(players[:3])
# ['charles', 'martina', 'michael']
print(players[2:])
['michael', 'florence', 'eli']
